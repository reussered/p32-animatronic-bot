GOBLIN HEAD FEATURES:
--------------------
- Large protruding brow ridge over deep eye sockets
- Jutting underbite jaw for aggressive appearance
- Long pointed ears extending from skull sides  
- Sharp angular cheekbones
- Skull ridge running front to back
- Forest green coloring recommended

ELECTRONICS:
-----------
- 2x GC9A01 displays in eyes (animated eyes)
- 1x GC9A01 display in mouth (expressions)
- 1x HC-SR04 sensor in nose (distance detection)
- ESP32-S3 controller with custom PCB

3D PRINT SETTINGS:
-----------------
Layer Height: 0.2mm
Infill: 15%
Supports: Yes (required for overhangs)
Material: PLA plastic
Color: Forest green #3D5A2B recommended

ASSEMBLY STEPS:
--------------
1. Print all STL files listed above
2. Install GC9A01 displays in eye and mouth mounts
3. Mount HC-SR04 sensor in nose bracket
4. Route all wiring through skull interior hollow space
5. Connect components to ESP32-S3 per wiring diagram
6. Test all displays and sensor before final assembly
7. Secure all mounting hardware with screws

COMPONENT POSITIONS (P32 Coordinate System):
------------------------------------------
Left Eye: -1.05 inches, +0.7 inches, -0.35 inches
Right Eye: +1.05 inches, +0.7 inches, -0.35 inches  
Nose Sensor: 0 inches, 0 inches, +0.25 inches
Mouth Display: 0 inches, -1.05 inches, 0 inches

All measurements from nose center as origin point.
